# CoDesign Tool — Website

Static website for the CoDesign Tool, a card-based co-design framework for cultural heritage.

Developed within the Horizon Europe PERCEIVE project (Grant 101061157) by CNR ISPC (Italy) and Fraunhofer IGD (Germany).

## Structure

```
site/
├── index.html          # Home
├── framework/          # Framework page
├── get-started/        # Get Started page
├── documentation/      # Documentation page
├── contact/            # Contact page
├── privacy/            # Privacy policy
├── cookies/            # Cookie policy
├── accessibility/      # Accessibility statement
├── legal/              # Legal notice
├── 404.html            # 404 page
├── _redirects          # Cloudflare Pages redirects
├── robots.txt
└── sitemap.xml
```

## Deploy

Connected to Cloudflare Pages. Every push to `main` triggers a new deploy.
